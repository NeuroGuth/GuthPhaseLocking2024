# GuthPhaseLocking2024
Analysis code for Guth et al., bioRxiv, 2024: "Theta-phase locking of single neurons during human spatial memory".

# TreasureHunt
This folder contains the scripts for all analyses in the paper.
All code is written in MATLAB R2023a (MathWorks, Natick, MA, USA).
Different folders contain the scripts for different analyses.
For example, the subfolder "PhaseAnalysis_20230921"
contains the script for the analysis of neuronal phase locking.

# Dependencies
The scripts in this repository use the following external functions/toolboxes:
[Fieldtrip](https://github.com/fieldtrip/fieldtrip), 
[WaveClus](https://github.com/csn-le/wave_clus), 
[Generalized Phase](https://github.com/mullerlab/generalized-phase), 
[CircStat](https://github.com/circstat/circstat-matlab), 
[SPRiNT](https://github.com/lucwilson/SPRiNT), 
[myBinomTest](https://de.mathworks.com/matlabcentral/fileexchange/24813-mybinomtest-s-n-p-sided), 
[cline](https://de.mathworks.com/matlabcentral/fileexchange/14677-cline), 
[rude](https://de.mathworks.com/matlabcentral/fileexchange/6436-rude-a-pedestrian-run-length-decoder-encoder), and 
[chi2test](https://de.mathworks.com/matlabcentral/fileexchange/16177-chi2test?requestedDomain=).
